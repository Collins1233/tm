# 🍭 The Treat Maker — Pre-Order System

A simple, beautiful pre-order system for **The Treat Maker** bakery.
Built with plain HTML, CSS and JavaScript — no server required.

---

## 📄 Pages

| File | Purpose | Who uses it |
|------|---------|-------------|
| `index.html` | Homepage — redirects to order page | Everyone |
| `treatmaker-order.html` | Pre-order form for customers | Customers |
| `treatmaker-admin.html` | Baker's dashboard to manage orders | Baker only |

---

## 🔗 Live Links (after publishing)

Once GitHub Pages is enabled, your site will be at:

- **Customer order page:** `https://YOUR-USERNAME.github.io/treatmaker/treatmaker-order.html`
- **Baker's dashboard:** `https://YOUR-USERNAME.github.io/treatmaker/treatmaker-admin.html`

> Replace `YOUR-USERNAME` with your actual GitHub username.

---

## 🚀 How to Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings**
3. Click **Pages** in the left sidebar
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save** — your site will be live in 1–2 minutes

## ⚙️ Features

- Customers can pre-order items with name, phone, delivery location & date
- Baker can open or close pre-orders with a toggle switch
- Baker can update each order status: New → Accepted → Ready → Done
- Export all orders as a CSV spreadsheet
- Works on any iPhone or Android browser — no app needed
- Fully offline-capable (orders stored in browser localStorage)
